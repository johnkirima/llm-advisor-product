## Recommendation Algorithm
- **Priority weights:** 0.50 (primary) / 0.30 (secondary) / 0.20 (tertiary)
- **Urgency adjustments:** urgent = +1.0 on reliability, relaxed = -1.0
- **Budget adjustments:** lowest-cost forced = +1.5 affordability, prefer-lower = +0.5; unlimited = +1.5 quality, no-limit = +0.5
- **Task-type bonus:** +0.5 if LLM's `strengthsFor[]` contains the task type
- Score = weighted sum → returns top 3 candidates sorted by final score
